# 简介
这是一个可以查询本地美食的对话机器人，另外也可以查询一些水果的知识。<br>
可以跟它说“我想吃火锅”、“苹果的热量”这些。<br>
项目是闲余时间所做，目前数据量较少。主要是人工整理，部分为使用机器阅读程序提取。
